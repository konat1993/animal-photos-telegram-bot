# Frontend

Next.js dashboard (map, filters, KPIs, species insights).

**Documentation for setup, environment variables, and commands lives in the [root README](../README.md#local-development).**

Quick reference:

```bash
cd frontend
cp .env.example .env.local
npm install
npm run dev
```

Use `NEXT_PUBLIC_DEMO_MODE=true` in `.env.local` to preview the UI without Supabase (see root README).
